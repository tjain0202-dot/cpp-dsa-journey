## About This Repository

This repository documents my journey of learning C++, Data Structures, and Algorithms.

All code implementations are written by me after working through the problems and reasoning about the solutions. I use AI as a learning mentor—not to copy solutions, but to understand concepts, identify mistakes, strengthen problem-solving, and discuss algorithm design. I intentionally try to derive implementations myself before asking for guidance.

Comments, explanations, and parts of the documentation may be refined with AI to improve clarity and make future revision easier.

The goal of this repository is to demonstrate my learning process, consistent practice, and growing understanding of DSA rather than simply collecting solved problems.
